### 如何获取凭证？

1. 登录 [Amazon Seller Central](https://sellercentral.amazon.com/)
2. 进入：**设置 > 用户权限 > 开发者中心**
3. 创建新应用程序
4. 获取 Client ID、Client Secret 和 Refresh Token

详细步骤：`config/amazon-spapi-setup.md`

### 失败情况

#### 认证失败
```
❌ 认证失败: invalid_client
```
**解决：** 检查 Client ID 和 Client Secret 是否正确

#### 凭证未配置
```
❌ 配置检查失败: amazon.spapi.clientID 未配置
```
**解决：** 在配置文件中填写真实凭证

#### API 速率限制
```
❌ 上传失败: API rate limit exceeded
```
**解决：** 等待 1-2 分钟后重试

## 🎯 命令行参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-sku` | 产品 SKU | `TEST-SKU-001` |
| `-title` | 商品标题 | `测试商品` |
| `-brand` | 品牌名称 | `TestBrand` |
| `-price` | 商品价格 | `19.99` |
| `-debug` | 调试模式 | `false` |

## ⚠️ 注意事项

1. **测试凭证**：配置文件中的凭证是示例，需要替换为真实凭证
2. **RefreshToken**：应该以 `Atzr|` 开头，长度通常超过 100 个字符
3. **ClientSecret**：是随机字符串，不是 `amzn1` 开头
4. **速率限制**：Amazon API 有速率限制，避免频繁请求
