# Amazon上架流程测试工具

## 🎯 功能说明

这个工具用于测试Amazon平台的完整产品上架流程，验证所有Handler是否正常工作。

## 🚀 使用方法

### 基本测试
```bash
go run cmd/amazon-upload-test/main.go
```

### 详细日志测试
```bash
go run cmd/amazon-upload-test/main.go -verbose
```

### 指定产品ID测试
```bash
go run cmd/amazon-upload-test/main.go -product "my-test-product-123"
```

### 指定配置文件测试
```bash
go run cmd/amazon-upload-test/main.go -config "config/config-prod.yaml"
```

## 🔧 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `-config` | `config/config-dev.yaml` | 配置文件路径 |
| `-product` | `test-product-001` | 测试产品ID |
| `-verbose` | `false` | 是否输出详细日志 |

## 🚨 注意事项

1. **配置文件**: 确保配置文件存在且格式正确
2. **网络连接**: 如果启用了LLM功能，需要网络连接
3. **API密钥**: 如果使用真实API，需要配置正确的密钥
4. **超时设置**: 默认超时5分钟，可根据需要调整
