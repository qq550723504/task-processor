# pkg 目录

## 用途

内部工具包，提供可复用的工具函数、类型定义和辅助功能。这些代码可以被项目内的任何模块使用。

## 目录结构

```
pkg/
├── amazon/       # Amazon 相关工具
├── downloader/   # 文件下载器
├── management/   # 管理系统客户端
├── mathutil/     # 数学工具
├── pricing/      # 价格处理
├── ptrutil/      # 指针工具
├── strutil/      # 字符串工具
├── types/        # 通用类型定义
└── utils/        # 通用工具函数
```

## 编码规范

1. 工具函数应该是纯函数，无副作用
2. 提供清晰的函数签名和文档
3. 充分的单元测试
4. 避免依赖具体的业务逻辑
