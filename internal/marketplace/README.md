# Marketplace Domain

This directory is the long-term home of marketplace-specific rules, models, APIs, and workspace behavior.

Each marketplace should group its own publishing, workspace, model, and API concerns together.

Approved marketplace roots:

- `internal/marketplace/shein`
- `internal/marketplace/amazon`
- `internal/marketplace/temu`
- `internal/marketplace/walmart`

Use this area for new platform-specific rules first, even if legacy implementations still exist elsewhere.
