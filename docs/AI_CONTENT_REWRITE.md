# AI内容重构功能说明

## 概述

AI内容重构功能使用OpenAI API自动优化TEMU产品的标题、描述和要点，使其更适合TEMU平台的展示和销售。

## 功能特点

### 1. 统一的产品定位

系统针对所有产品使用统一的重构策略，聚焦于成人和通用使用场景。

**核心定位：**
- 不上架儿童相关产品
- 所有产品文案避免儿童相关描述
- 聚焦于成人或通用使用场景
- 使用专业、成熟的表达方式

### 2. 英文输出强制要求

🚨 **关键特性：系统强制要求AI生成英文内容**

- ✅ 只使用英文字母（A-Z, a-z）、数字（0-9）和基本标点符号
- ❌ 不使用中文、日文、韩文或其他非英文字符
- 🎯 确保内容能通过TEMU平台的字符验证
- 📝 提示词中多次强调"ENGLISH ONLY"

**为什么需要英文？**
- TEMU平台的ProductNameValidator和ProductDescriptionValidator会过滤掉中文字符
- 如果AI生成中文内容，会被验证器清除，导致标题和描述为空或只剩"-"

### 3. AI重构策略

✅ **重点突出：**
- 产品的实用性和功能性
- 适用场景和用途
- 材质和工艺
- 性价比和品质
- 用户体验

⚠️ **重要约束：**
- 不添加儿童相关的描述
- 不提及"适合儿童"、"儿童专用"、"宝宝"、"孩子"、"kids"、"children"等词汇
- 即使原Amazon产品信息中包含儿童相关内容，也会改写为通用或成人使用场景
- 使用专业、成熟的表达方式

### 4. 内容重构标准

#### 标题重构
- **长度**：20-200字符
- **内容**：包含核心关键词和产品类型
- **风格**：突出主要特性或卖点
- **规范**：避免特殊符号和装饰字符

#### 描述重构
- **长度**：200-2000字符
- **结构**：清晰分段，逻辑合理
- **内容**：产品特性、用途、优势
- **表达**：具体描述，避免空洞形容词

#### 要点重构
- **数量**：3-6个要点
- **长度**：每个15-120字符
- **内容**：突出不同特性
- **排序**：按重要性排列

## 在Pipeline中的位置

AI内容重构处理器位于pipeline的第20步，在BuildSpuHandler之后：

```
18. 模板查询
19. 构建SPU（从Amazon设置初始标题、描述、要点）
20. AI内容重构 ← 在这里重构内容
21. 产品名称验证和清理
22. 产品要点验证和优化
23. 产品描述验证和优化
24. 清除品牌名称
```

这个位置确保：
1. BuildSpuHandler已经从Amazon产品设置了初始的标题、描述和要点
2. AI内容重构可以基于这些初始内容进行优化
3. 后续的验证器可以对AI生成的内容进行规范化处理
4. 品牌清除在最后执行，确保所有内容都不包含品牌名

## 配置要求

### OpenAI配置

需要在配置文件中设置OpenAI相关参数：

```yaml
openai:
  api_key: "your-api-key"
  model: "gpt-4"  # 或 gpt-3.5-turbo
  base_url: "https://api.openai.com/v1"  # 可选，使用自定义API端点
  timeout: 60  # 超时时间（秒）
```

### 禁用AI重构

如果不想使用AI重构功能，可以：

1. 不配置OpenAI参数（系统会自动跳过）
2. 或在pipeline中注释掉该处理器

## 使用示例

### 示例1：办公椅

**输入（Amazon）**
```
标题: Ergonomic Office Chair with Lumbar Support High Back Executive Chair
描述: This ergonomic office chair features adjustable lumbar support...
```

**输出（TEMU）**
```
标题: 人体工学办公椅 高背老板椅 带腰部支撑 可调节电脑椅
描述: 这款人体工学办公椅采用高品质材料制作，配备可调节腰部支撑系统...
      适合长时间办公使用，有效缓解腰背疲劳，提升工作效率...
要点:
- 人体工学设计：符合人体曲线，长时间使用不疲劳
- 腰部支撑：可调节腰托，有效缓解腰部压力
- 高品质材料：加厚海绵坐垫，透气网布靠背，经久耐用
```

### 示例2：原Amazon产品包含儿童关键词的处理

**输入（Amazon）**
```
标题: Kids Building Blocks Set 100 Pieces Educational Toy for Children Ages 3+
描述: This colorful building blocks set includes 100 pieces for children...
```

**输出（TEMU）**
```
标题: 创意积木套装 100块 益智拼装玩具 多功能组合
描述: 这款创意积木套装包含100块色彩鲜艳的积木，采用优质ABS材质制作...
      适合家庭娱乐、创意拼装，帮助培养空间想象力和动手能力...
要点:
- 优质材质：采用环保ABS塑料，安全耐用
- 多样玩法：100块积木可自由组合，发挥创造力
- 家庭娱乐：适合家庭聚会、休闲时光，增进互动
```

注意：即使原产品信息中包含"Kids"、"Children"等儿童关键词，AI也会将其改写为通用场景描述，完全避免儿童相关词汇。

## 错误处理

如果AI重构失败（如API调用失败、网络问题等），系统会：

1. 记录警告日志
2. 保持原始内容不变
3. 继续执行后续处理器
4. 不会中断整个pipeline

这确保了即使AI服务不可用，产品发布流程仍然可以正常进行。

## 性能考虑

- **API调用时间**：通常2-10秒，取决于模型和内容长度
- **重试机制**：内置3次重试，使用指数退避策略
- **超时设置**：默认60秒超时
- **并发控制**：建议根据OpenAI API限制调整并发数

## 最佳实践

1. **使用GPT-4**：获得更好的重构质量（成本较高）
2. **使用GPT-3.5-turbo**：平衡质量和成本（推荐）
3. **监控日志**：定期检查AI重构的质量
4. **人工审核**：对重要产品进行人工审核
5. **A/B测试**：对比AI重构前后的转化率
