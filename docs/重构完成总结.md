**文件：** 
- `internal/crawler/amazon/browser/zipcode_input_handler.go` (重构)
- `internal/crawler/amazon/browser/zipcode_strategy.go` (新建)
- `internal/crawler/amazon/browser/zipcode_strategy_city_dropdown.go` (新建)
- `internal/crawler/amazon/browser/zipcode_strategy_japanese.go` (新建)
- `internal/crawler/amazon/browser/zipcode_strategy_standard.go` (新建)

**问题：**
- 原文件536行，包含3个高复杂度函数（复杂度27、23、20）
- 多个国家/地区的邮编处理逻辑混在一起
- `handleCityDropdown` 复杂度27（建议<15）
- 难以测试和维护

**重构方案：**
使用策略模式，按国家/地区拆分处理逻辑。

**架构设计：**

```
ZipcodeInputHandler (主处理器)
    ├── strategies []ZipcodeStrategy (策略列表)
    │   ├── CityDropdownStrategy (沙特、阿联酋)
    │   ├── JapaneseZipcodeStrategy (日本)
    │   └── StandardZipcodeStrategy (美国、欧洲等)
    │
    └── handleZipcodeInput() (策略选择器)
```

**重构前：**
```go
// zipcode_input_handler.go - 536行
func (zih *ZipcodeInputHandler) handleZipcodeInput(...) error {
    // 尝试城市下拉框 (复杂度27)
    if handled, err := zih.handleCityDropdown(...); handled {
        return nil
    }
    
    // 尝试日本站 (复杂度20)
    if handled, err := zih.handleJapaneseZipcode(...); handled {
        return nil
    }
    
    // 标准输入框
    return zih.handleStandardZipcode(...)
}
```

**重构后：**
```go
// zipcode_input_handler.go - 主处理器（简化）
type ZipcodeInputHandler struct {
    strategies []ZipcodeStrategy
}

func (zih *ZipcodeInputHandler) handleZipcodeInput(...) error {
    for _, strategy := range zih.strategies {
        if strategy.CanHandle(page, zipcode) {
            return strategy.Handle(page, zipcode)
        }
    }
    return fmt.Errorf("没有找到合适的策略")
}

// zipcode_strategy.go - 策略接口
type ZipcodeStrategy interface {
    CanHandle(page playwright.Page, zipcode string) bool
    Handle(page playwright.Page, zipcode string) error
    GetName() string
}

// zipcode_strategy_city_dropdown.go - 城市下拉框策略
type CityDropdownStrategy struct {
    BaseZipcodeStrategy
}
// 所有沙特、阿联酋相关逻辑

// zipcode_strategy_japanese.go - 日本站策略
type JapaneseZipcodeStrategy struct {
    BaseZipcodeStrategy
}
// 所有日本站相关逻辑

// zipcode_strategy_standard.go - 标准策略
type StandardZipcodeStrategy struct {
    BaseZipcodeStrategy
}
// 所有标准输入框逻辑
```
