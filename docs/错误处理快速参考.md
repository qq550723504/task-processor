## 📝 常用操作

### 创建错误
```go
// 简单错误
err := errors.New(errors.ErrCodeValidation, "产品ID不能为空")

// 格式化错误
err := errors.Newf(errors.ErrCodeValidation, "产品ID无效: %s", productID)
```

### 包装错误
```go
// 包装现有错误
err := errors.Wrap(err, errors.ErrCodeNetwork, "网络请求失败")

// 格式化包装
err := errors.Wrapf(err, errors.ErrCodeExternalAPI, "调用API失败: url=%s", url)
```

### 判断错误
```go
// 判断错误码
if errors.IsCode(err, errors.ErrCodeNetwork) {
    // 处理网络错误
}

// 判断是否可重试
if errors.IsRetryable(err) {
    // 重试操作
}

// 判断是否关键错误
if errors.IsCritical(err) {
    // 终止操作
}
```

## 🔄 重试机制

### 基本重试
```go
ctx := context.Background()
config := errors.DefaultRetryConfig()

err := errors.Retry(ctx, config, func() error {
    return doSomething()
})
```

### 自定义重试配置
```go
config := errors.RetryConfig{
    MaxRetries:    5,                    // 最大重试5次
    InitialDelay:  2 * time.Second,      // 初始延迟2秒
    MaxDelay:      30 * time.Second,     // 最大延迟30秒
    BackoffFactor: 2.0,                  // 指数退避因子
    RetryableCheck: errors.IsRetryable,  // 重试判断函数
}

err := errors.Retry(ctx, config, func() error {
    return callAPI()
})
```

## 🎯 错误处理器

### 创建和使用
```go
// 创建错误处理器
handler := errors.NewDefaultErrorHandler(logger)

// 处理错误
err := handler.Handle(err)

// 判断是否重试
if handler.ShouldRetry(err) {
    // 重试逻辑
}

// 判断是否终止

```

## 💡 最佳实践

### ✅ 推荐做法

```go
// 1. 使用具体的错误码
return errors.New(errors.ErrCodeValidation, "产品ID不能为空")

// 2. 包装底层错误
return errors.Wrap(err, errors.ErrCodeNetwork, "网络请求失败")

// 3. 使用错误处理器
err = handler.Handle(err)

// 4. 使用重试机制
errors.Retry(ctx, config, func() error { ... })

// 5. 安全启动goroutine
errors.SafeGo(func() { ... }, logger)
```

### ❌ 避免做法

```go
// 1. 不要在非main函数中使用Fatal
log.Fatal("错误") // ❌

// 2. 不要在运行时panic
panic("错误") // ❌

// 3. 不要丢失原始错误
return errors.New(errors.ErrCodeSystem, "错误") // ❌ 丢失了原始错误

// 4. 不要重复记录错误
logger.Error(err)
return handler.Handle(err) // ❌ 重复记录

// 5. 不要使用通用错误码
return errors.New(errors.ErrCodeSystem, "验证失败") // ❌ 应该用 ErrCodeValidation
```

## 🔍 常见场景

### 场景1: API调用
```go
func (c *Client) CallAPI(ctx context.Context) error {
    config := errors.DefaultRetryConfig()
    config.MaxRetries = 3
    
    return errors.Retry(ctx, config, func() error {
        resp, err := c.httpClient.Get(url)
        if err != nil {
            return errors.Wrap(err, errors.ErrCodeNetwork, "HTTP请求失败")
        }
        return nil
    })
}
```

### 场景2: 数据验证
```go
func (s *Service) ValidateProduct(product *Product) error {
    var errs []error
    
    if product.ID == "" {
        errs = append(errs, errors.New(errors.ErrCodeValidation, "产品ID不能为空"))
    }
    
    if product.Price <= 0 {
        errs = append(errs, errors.New(errors.ErrCodeValidation, "产品价格必须大于0"))
    }
    
    return errors.Combine(errs...)
}
```

### 场景3: 后台任务
```go
func (s *Service) StartBackgroundTask() {
    errors.SafeGo(func() {
        for {
            if err := s.processTask(); err != nil {
                s.errorHandler.Handle(err)
            }
            time.Sleep(time.Minute)
        }
    }, s.logger)
}
```

### 场景4: 构造函数
```go
func NewService(config *Config) (*Service, error) {
    if config == nil {
        return nil, errors.New(errors.ErrCodeConfig, "配置不能为空")
    }
    
    client, err := createClient(config)
    if err != nil {
        return nil, errors.Wrap(err, errors.ErrCodeSystem, "创建客户端失败")
    }
    
    return &Service{
        config: config,
        client: client,
    }, nil
}
```
