# SHEIN 英文描述 AI 回退设计

## 目标

移除固定英文营销描述回退。商品已有合格英文描述时保留原文；描述缺失或包含中文时，必须由 AI 生成英文描述。

## 行为

1. `canonical.Description` 为非空且不含中文：直接使用该描述，不调用 AI。
2. 描述为空或包含中文：调用既有文本生成服务，输入商品标题、类目、材质、规格和已有描述。
3. AI 返回为空、包含中文、格式无效，或调用发生错误/超时：返回可识别的描述生成失败，并让 ListingKit/SHEIN 流程进入 `needs_review` 阻断状态。
4. 不再保留或调用 `synthesizeEnglishDescription`；绝不以固定句子替代失败结果。

## 边界

- 不覆盖已有合格英文描述。
- 不改变标题生成策略。
- 阻断信息必须可在审核重点中展示，且最终提交不可继续。
- 覆盖已有英文描述、AI 成功生成、AI 失败阻断、AI 返回无效内容四类自动化测试。
