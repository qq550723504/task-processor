# SHEIN 同步商品 SKU 价格表格设计

## 目标

同步商品页按 SKU 展开价格信息：每个 SKU 占一行，同步售价与 SHEIN 供货价分别显示在独立列中。

## 约束

- 商品信息、主规格、销量、成本/利润、库存、上架状态与时间均为商品级信息，只在 SKU 组的首行显示，并跨越该商品的全部 SKU 行。
- 以同步售价快照的 SKU 顺序为主；仅存在于供货价快照中的 SKU 追加到末尾。
- SKU 行按 SKU 编码合并同步售价与 SHEIN 供货价，不能再显示两段重复的 SKU 列表。
- 缺失同步售价显示“—”；尚未获得供货价快照时显示“待同步”。
- 成本与利润仍使用既有商品级快照，不把供货价误作为维护成本。

## 非目标

- 不修改同步接口、快照存储或活动报名的价格计算。
- 不补充 SKU 属性解析；本次仅改善已同步 SKU 价格的展示方式。
