## 📊 配置迁移指南

### 自动迁移
现有的配置文件会自动迁移，无需手动修改。系统会：
1. 从 `browser` 配置读取浏览器相关设置
2. 自动复制到 `amazon` 配置的兼容性字段中
3. 保证现有代码正常工作

### 推荐迁移
为了获得最佳体验，建议手动迁移配置文件：

1. **将浏览器配置移动到 `browser` 部分**
2. **简化 `amazon` 配置，只保留 Amazon 特有配置**
3. **验证配置文件语法正确性**
4. **测试应用程序正常启动**
