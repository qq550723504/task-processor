# 日志迁移示例

本文档提供实际的代码迁移示例，展示如何将项目中的日志代码迁移到新的统一日志系统。

#### 迁移后
```go
package temu

import (
	"context"
	"task-processor/internal/core/logger"
	"github.com/sirupsen/logrus"
)

type TaskHandler struct {
	processor *TemuProcessor
	logger    *logrus.Entry
	helper    *logger.LoggerHelper
}

func NewTaskHandler(processor *TemuProcessor) *TaskHandler {
	log := logger.GetGlobalLogger("task_handler").WithFields(logrus.Fields{
		logger.FieldComponent: "task_handler",
		logger.FieldPlatform:  "temu",
	})
	
	return &TaskHandler{
		processor: processor,
		logger:    log,
		helper:    logger.NewLoggerHelper(log),
	}
}

func (h *TaskHandler) ProcessTask(ctx context.Context, task model.Task, executor *TemuPipelineExecutor) error {
	// 使用辅助方法记录任务开始
	h.helper.LogTaskStart(task.ID, task.ProductID)
	
	// 使用 LogOperation 自动记录耗时
	err := h.helper.LogOperation("process_task", func() error {
		return executor.Execute(taskCtx)
	})
	
	if err != nil {
		h.handleTaskFailure(task, err)
		return err
	}
	
	return nil
}

func (h *TaskHandler) updateTaskStatusSync(taskID int64, status, errorMsg string) {
	h.logger.WithFields(logrus.Fields{
		logger.FieldTaskID: taskID,
		logger.FieldStatus: status,
	}).Debug("准备同步更新任务状态")
	
	// ... 更新逻辑
	
	h.logger.WithFields(logrus.Fields{
		logger.FieldTaskID: taskID,
		logger.FieldStatus: status,
	}).Info("任务状态同步更新成功")
}
```

## 示例 2: TaskSubmitter 迁移

### 文件：internal/platforms/temu/task_submitter.go

#### 迁移前
```go
package temu

import (
	"github.com/sirupsen/logrus"
)

func (s *TemuTaskSubmitter) SubmitTask(task model.Task) error {
	if task.CreateTime > 0 {
		waitTime := time.Since(time.Unix(task.CreateTime/1000, 0))
		logrus.Infof("[TEMU] Task %d: Pending -> Processing (Priority: %d, WaitTime: %v)",
			task.ID, task.Priority, waitTime.Truncate(time.Millisecond))
	}
	
	if err := s.workerPool.Submit(task); err != nil {
		return err
	}
	
	logrus.Infof("[TEMU] ✅ 任务已提交到工作池: ID=%d, ProductID=%s", task.ID, task.ProductID)
	return nil
}
```

#### 迁移后
```go
package temu

import (
	"task-processor/internal/core/logger"
	"github.com/sirupsen/logrus"
)

type TemuTaskSubmitter struct {
	workerPool worker.WorkerPool
	logger     *logrus.Entry
}

func NewTemuTaskSubmitter(workerPool worker.WorkerPool) *TemuTaskSubmitter {
	return &TemuTaskSubmitter{
		workerPool: workerPool,
		logger:     logger.GetGlobalLogger("temu_task_submitter"),
	}
}

func (s *TemuTaskSubmitter) SubmitTask(task model.Task) error {
	fields := logrus.Fields{
		logger.FieldTaskID:    task.ID,
		logger.FieldProductID: task.ProductID,
		logger.FieldPlatform:  "temu",
		"priority":            task.Priority,
	}
	
	if task.CreateTime > 0 {
		waitTime := time.Since(time.Unix(task.CreateTime/1000, 0))
		fields["wait_time_ms"] = waitTime.Milliseconds()
		s.logger.WithFields(fields).Info("任务状态变更: Pending -> Processing")
	}
	
	if err := s.workerPool.Submit(task); err != nil {
		s.logger.WithError(err).WithFields(fields).Error("任务提交失败")
		return err
	}
	
	s.logger.WithFields(fields).Info("任务已提交到工作池")
	return nil
}
```

## 示例 6: 错误处理迁移

### 文件：internal/platforms/shein/repo/client/error_handler.go

#### 迁移前
```go
package client

import "fmt"

func (h *ErrorHandler) logError(operation string, err error) {
	// 这里可以添加日志记录逻辑
	// 例如使用 logrus 或其他日志库
	fmt.Printf("API错误 [%s]: %v\n", operation, err)
}
```

#### 迁移后
```go
package client

import (
	"task-processor/internal/core/logger"
	"github.com/sirupsen/logrus"
)

type ErrorHandler struct {
	logger *logrus.Entry
}

func NewErrorHandler() *ErrorHandler {
	return &ErrorHandler{
		logger: logger.GetGlobalLogger("shein_error_handler"),
	}
}

func (h *ErrorHandler) logError(operation string, err error) {
	h.logger.WithError(err).WithField(logger.FieldOperation, operation).Error("API错误")
}
```

## 迁移检查清单

完成每个文件的迁移后，使用此清单验证：

- [ ] 导入了 `task-processor/internal/core/logger`
- [ ] 移除了 `logrus.Info()` 等全局调用
- [ ] 移除了 `fmt.Printf()` 调试输出
- [ ] 结构体中添加了 `logger *logrus.Entry` 字段
- [ ] 构造函数中使用 `logger.GetGlobalLogger()` 初始化
- [ ] 字段命名使用 `logger.Field*` 常量
- [ ] 错误日志使用 `WithError()` 方法
- [ ] 考虑使用 `LoggerHelper` 简化常见操作
- [ ] 测试代码使用统一的日志配置

## 批量迁移脚本

```bash
#!/bin/bash

# 查找需要迁移的文件
echo "查找使用 logrus 全局方法的文件..."
grep -rl "logrus\.(Info\|Error\|Warn\|Debug)" --include="*.go" . | grep -v vendor | grep -v ".git"

echo ""
echo "查找使用 fmt.Printf 的文件..."
grep -rl "fmt\.Printf\|fmt\.Println" --include="*.go" . | grep -v vendor | grep -v ".git" | grep -v "_test.go"

echo ""
echo "查找字段命名不规范的文件..."
grep -rl "\"taskId\"\|\"TaskID\"\|\"productId\"" --include="*.go" . | grep -v vendor | grep -v ".git"
```
