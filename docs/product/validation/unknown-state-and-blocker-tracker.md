## 使用规则

- `open`：已发现但还没有稳定映射、兜底展示或 SOP。
- `mapped`：后端 / 前端 / SOP 已经有明确映射，但还需要下一轮真实任务验证。
- `closed`：已经用真实 run 证明用户可理解、可恢复或可升级。
- `won't fix`：明确不支持，且 UI / SOP 已说明原因。
