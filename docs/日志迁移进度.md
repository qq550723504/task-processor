## 迁移检查清单

每个文件迁移后需要检查：

- [ ] 导入了 `task-processor/internal/core/logger`
- [ ] 移除了 `logrus.Info()` 等全局调用
- [ ] 移除了 `fmt.Printf()` 调试输出
- [ ] 结构体中添加了 `logger *logrus.Entry` 字段
- [ ] 构造函数中使用 `logger.GetGlobalLogger()` 初始化
- [ ] 字段命名使用 `logger.Field*` 常量
- [ ] 错误日志使用 `WithError()` 方法
- [ ] 代码可以编译通过
- [ ] 相关测试通过
