# 材质一致性问题修复

## 问题描述

TEMU平台审核时发现产品的材质信息前后不一致，导致审核失败：

**错误信息示例**：
```
商品描述的材质前后不符
标题仅包含钢材描述，且该描述与属性栏中的金属/钢材属性明显不符
请修改违规信息后重新提交
请更正违规宣传内容，更新商品信息，并等待平台审核
```

**问题原因**：
- 产品标题中明确提到"Steel"（钢材）
- 但AI在选择材质属性时，由于提示词指导不当，选择了"混合材质"或其他不匹配的材质
- 导致标题描述与属性栏的材质信息不一致

## 根本原因分析

在`platforms/temu/handlers/ai_prompt_builder.go`的提示词中，对材质属性的指导过于笼统：

```go
// 原来的提示词（有问题）
- 材质（必选）：选择"混合材质"、"其他"、"复合材质"
```

这个指导会导致AI在不确定时总是选择"混合材质"，而不是根据产品标题和描述中的实际材质信息来选择。

## 解决方案

### 1. 更新系统提示词 - 添加材质一致性原则

在映射规则中添加了第7条"材质一致性原则"：

```go
7. **材质一致性原则**（🚨 极其重要，违反会导致审核失败）：
   - 材质属性必须与产品标题/描述中的材质描述完全一致
   - 如果标题提到"Steel"、"钢"、"不锈钢"，材质必须选择"钢"、"不锈钢"或"金属"
   - 如果标题提到"Plastic"、"塑料"，材质必须选择"塑料"相关选项
   - 如果标题提到"Wood"、"木"，材质必须选择"木质"相关选项
   - 只有在完全无法从标题/描述中确定材质时，才选择"混合材质"或"其他"
```

### 2. 更新用户提示词 - 详细的材质选择策略

在属性选择策略的第3步中，将材质的指导从简单的"选择混合材质"改为：

```go
- 材质（必选）：⚠️ 重要！必须与标题/描述中的材质描述保持一致
  * 如果标题/描述明确提到材质（如"钢材"、"不锈钢"、"塑料"、"木质"等），必须选择对应的材质选项
  * 如果标题提到"Steel"，必须选择"钢"、"不锈钢"或"金属"相关选项，不能选"混合材质"
  * 如果标题提到"Plastic"，必须选择"塑料"相关选项
  * 如果标题提到"Wood"，必须选择"木质"相关选项
  * 只有在完全无法确定材质时，才选择"混合材质"、"其他"、"复合材质"
```

### 3. 添加输出检查清单

在输出检查部分添加了材质一致性检查：

```go
✅ 【关键】材质一致性检查：
   - 检查标题中是否提到材质关键词（Steel/钢/不锈钢/Plastic/塑料/Wood/木/等）
   - 如果标题提到"Steel"或"钢"，材质属性必须选择"钢"、"不锈钢"或"金属"相关选项
   - 材质选择必须与标题/描述中的材质描述完全一致
   - 不要在标题明确提到具体材质时选择"混合材质"或"其他"
```

### 4. 强调核心原则

在核心原则中添加了材质一致性的强调：

```go
- 🚨🚨 材质属性必须与标题/描述保持一致，避免审核失败！
  例如：标题含"Steel"→必选"钢"/"不锈钢"/"金属"，不能选"混合材质"
```

## 修改的文件

- `platforms/temu/handlers/ai_prompt_builder.go` - 更新AI提示词

## 预期效果

修复后，AI在选择材质属性时会：

1. **优先从标题中提取材质信息**
   - 标题含"Steel" → 选择"钢"、"不锈钢"或"金属"
   - 标题含"Plastic" → 选择"塑料"
   - 标题含"Wood" → 选择"木质"

2. **其次从描述和产品详情中提取**
   - 检查产品描述中的材质说明
   - 检查Amazon产品详情中的Material字段

3. **最后才使用中性值**
   - 只有在完全无法确定材质时，才选择"混合材质"或"其他"

## 示例对比

```
产品标题: "Stainless Steel Kitchen Organizer Rack"
AI选择的材质: "混合材质" ❌

```

```

结果: TEMU审核通过
```

## 注意事项

1. **材质一致性是TEMU审核的重点**，违反会直接导致审核失败
2. **标题中的材质描述优先级最高**，必须与属性栏保持一致
3. **不要过度使用"混合材质"**，只有在真正无法确定时才使用
4. **如果产品确实是多种材质组成**，可以选择主要材质或"混合材质"，但要确保标题中也有相应的描述

## 测试建议

在部署后，建议测试以下场景：

1. 标题明确提到单一材质（如"Steel Rack"）
2. 标题提到多种材质（如"Wood and Metal Shelf"）
3. 标题未提到材质但描述中有材质信息
4. 标题和描述都没有明确的材质信息

确保AI在各种场景下都能做出正确的材质选择。
